processAccRaw

  • processAccRaw(mac: string, raw: string, rssi: string, time: string): AccRawProcessedType

  • PROCESSAMENTO DOS DADOS DO SERVIÇO DE ACC RAW

    Os dados de Acc Raw podem conter dados de apenas um eixo ou dados de todos os eixos. Desta forma a propriedade accRaw que é recebida na resposta deste método é um vetor podendo conter uma ou três posições dependendo da configuração do eixo de coleta.

    Exemplo para um único eixo:

    {
  axis: "x"
  accRaw: [["Conteúdo do Eixo X"]]
}

    Exemplo para todos os eixos:

    {
  axis: "all"
  accRaw: [["Conteúdo do Eixo X"], ["Conteúdo do Eixo Y"], ["Conteúdo do Eixo Z"]]
}

    Parameters

    • mac: string
    • raw: string
    • rssi: string
    • time: string

    Returns AccRawProcessedType
